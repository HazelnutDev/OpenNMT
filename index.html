    Tentang Proyek

    Proyek ini bertujuan membangun model terjemahan mesin dari Bahasa Indonesia ke Bahasa Bima (dan sebaliknya) menggunakan framework OpenNMT-py.

    Fokus utama proyek adalah: - Mempreservasi bahasa daerah Bima melalui teknologi NLP.
    - Menyediakan dataset paralel Indonesia â†” Bima.
    - Menghadirkan model yang dapat dijalankan secara lokal.

    Teknologi yang Digunakan

    • Python 3.8+
    • OpenNMT-py v1 (stabil, berbasis CLI)
    • SentencePiece (untuk subword tokenization)
    • PyTorch sebagai backend
